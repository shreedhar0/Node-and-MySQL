# Node-and-MySQL
Basic node and MySQL code 


As you clone this project you need to setup an environment for it. Following are some steps to do :
First install nodejs in your system.
Open project directory in terminal or command prompt.
Initialize node in this directory by typing following command -  

node init -y
npm install

In terminal type - 

npm start

Open your browser and go to localhost -

localhost:3000/

And you are done!!
